# Grownunited-Internship

This repository contains a simple Python-based Student Management System project created for the Grownunited internship.

## Project structure

- `student/Main.py` - main menu-driven application
- `student/Student.py` - student data class
- `student/StudentManager.py` - student record manager and file persistence
- `student/students.txt` - saved student records

## How to run

```bash
python3 student/Main.py
```

## Notes

Save data before exiting to keep student records in `students.txt`.
